Welcome to Chat Backend Test UI

This UI allows you to test all chat functionality:

  • 1-on-1 Chat: Direct messaging between users
  • Group Chat: Multi-user conversations
  • Business Chat: Customer support conversations
  • Real-time: WebSocket connections for live messaging
  • Persistence: All data stored in PostgreSQL
  • Caching: User profiles cached in Redis

Start by authenticating with your JWT token, then create or select a conversation.
